## 子代理任务包

```text
任务编号与目标：
项目/工作区路径：
基准提交：
对应需求/场景：
非目标：
允许文件/符号：
禁止范围：
已集成依赖：
必要代码事实与项目约束：
失败测试或替代验证：
验证命令与预期：
禁止事项：
- 不读取完整聊天记录
- 不重新解释需求范围
- 不修改权威 tasks/状态源
- 不修改无关文件
- 未经授权不提交、合并或推送

返回状态（机读 / 中文）：
- `completed` / 已完成
- `completed_with_concerns` / 完成但有疑虑
- `needs_context` / 需要上下文
- `blocked` / 被阻塞

返回内容：实际修改、验证证据、疑虑/阻塞、范围偏差
```
